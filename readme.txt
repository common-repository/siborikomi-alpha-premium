This is csv checkbox search plugin. You can mount a checkbox parts in your website to search csv data.

== Description ==

This csv checkbox search plugin can that you mount a checkbox parts in your website to search csv data.
The necessary action is very simple.
1. To prepare a csv file taking the csv file template in plugin folder.
2. Get plugin key in my site.
3. Upload the csv file and setting details if you need.

It is not necessary that you write any program.

== Installation ==

You can install this plugin directly from your WordPress dashboard:

e.g.

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Csv Checkbox Search screen to configure the plugin
4. Upload csv file and setting details.
